四大核心組件
IADL
Industrial Asset Description Language
標準化資產定義語言，涵蓋 3D 幾何、數據標籤、行為模型、MEP 系統，實現「一次建模，到處使用」
FDL
Factory Design Language
聲明式工廠設計語言，描述資產的佈局、數量、位置和關係，支援分散式 Worker 分配提示
NDH
Neutral Data Hub
中立數據中樞，管理 Asset Servants 生命週期，整合 OT/IT 數據源，支援可插拔時序數據庫
Omniverse
NVIDIA Omniverse Connector
雙向連接器，實現與 NVIDIA Omniverse 的即時同步，支援多用戶協作和高保真 3D 渲染
